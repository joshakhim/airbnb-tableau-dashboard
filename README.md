# 🏡 Airbnb Tableau Dashboard Project

## 📌 Project Overview

This project analyzes Airbnb listing data using Tableau to identify profitable locations and seasonal revenue trends for potential Airbnb investments.

The goal was to answer business questions such as:
- Which zip codes generate the highest Airbnb prices?
- What property sizes perform best?
- Which time of year generates the most revenue?
- Where would be the best place to build an Airbnb property?

---

# 🛠 Tools Used

- Tableau
- Excel / CSV
- Data Visualization
- Dashboard Design

---

# 📊 Dashboard Features

- Average price per bedroom analysis
- Revenue trends over time
- Zip code pricing comparison
- Interactive geographic visualization
- Listing distribution insights

---

# 📈 Key Insights

- Larger bedroom properties generated higher average prices.
- Certain zip codes consistently outperformed others in pricing potential.
- Revenue trends fluctuated throughout the year, revealing stronger booking periods.
- Geographic visualization helped identify high-value Airbnb locations.
